This python script essentially allows you to encrypt or decrypt a file of your choice using the following commands:

python encryptdecrypt.py <encrypt | decrypt> <filename> <passphrase>

REQUIRED PYTHON LIBRARIES:

1) simple-crypt (https://pypi.python.org/pypi/simple-crypt)
2) pycrypto (https://pypi.python.org/pypi/pycrypto)
